
PetCare App – Base de Datos

Este repositorio contiene el diseño y los scripts de la base de datos para la aplicación móvil **PetCare App**, una plataforma que conecta dueños de mascotas con cuidadores (paseo, cuidado, asistencia) al estilo inDrive.

Entidades principales

El modelo relacional cubre los siguientes conceptos:

- Usuarios (dueños y cuidadores)
- Roles (dueño, cuidador)
- Mascotas
- Solicitudes de servicio (publicadas por dueños)
